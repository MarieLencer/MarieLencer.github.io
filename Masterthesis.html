Masterthesis: Spirit Garden

Project Description

Based on the Spirit Garden project this masterthesis focuses on the development of a personality system for the creatures. Using the well studied psychological model for personality HEXACO I created a prototype for a personality system. For this purpose I used my own implementation of a behaviour tree and developed a system to define individual personality profiles. The behaviour tree allows developers to define variations of a behaviour and tie them to certain personality traits. The personality profile of a creature will then determine which reaction will be shown in a situation. The players will have to adjust their actions to the personality of creature to gain it's trust. This allows for more varied gameplay and more interesting situations.

This project was created as my masterthesis for my game design master's degree at the Hochschule für Technik und Wirtschaft Berlin.
